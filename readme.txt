一个受KAM.SPACE启发的极简主义作品集主题，集成微信朋友圈功能。

== 描述 ==
KAM Inspired 是一个现代化的WordPress主题，具有以下特点：

- 微信朋友圈风格的首页布局
- 极简主义设计风格
- 完全响应式布局
- 实时点赞评论互动
- 图片预览功能
- QQ号快速留言系统
- 作品集自定义文章类型
- 完整的后台自定义设置

== 安装 ==
1. 上传 kam-inspired-theme 文件夹到 /wp-content/themes/
2. 在WordPress后台激活主题
3. 前往 外观 > 自定义 进行主题设置
4. 设置朋友圈背景和用户信息
5. 创建菜单：外观 > 菜单

== 页面模板 ==
主题包含以下页面模板：
- 朋友圈首页 (index.php)
- 关于页面 (about.php)
- 联系页面 (contact.php)
- 默认页面模板

== 功能特点 ==
= 朋友圈功能 =
- 微信朋友圈风格布局
- 动态发布界面
- 点赞和评论互动
- 图片网格展示
- 图片预览模态框
- 加载更多功能
- 实时用户交互

= 自定义设置 =
- Logo上传
- 颜色定制
- 朋友圈背景设置
- 用户信息设置
- 社交媒体链接
- 联系信息设置

= QQ留言系统 =
- 只需输入QQ号即可留言
- 自动获取QQ昵称和头像
- 实时验证和缓存

== 使用说明 ==
1. 朋友圈设置：在外观 > 自定义 > 朋友圈设置中配置
2. 发布动态：使用"动态"文章类型发布内容
3. 用户互动：前台支持点赞和评论
4. 图片管理：支持多图片网格布局
